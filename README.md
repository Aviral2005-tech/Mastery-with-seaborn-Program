# Mastery-with-seaborn-Program

## Project Overview

The **Interactive Sales Dashboard** is a Python-based data visualization project that analyzes sales data and presents meaningful business insights through static and interactive dashboards. The project uses **Pandas** for data manipulation, **Seaborn** and **Matplotlib** for statistical visualizations, and **Plotly** for interactive charts.

The dashboard helps analyze sales trends, customer behavior, regional performance, and product-wise revenue using professional visualizations and interactive reports.

---

# Project Objectives

- Load and preprocess sales data.
- Perform exploratory data analysis (EDA).
- Generate statistical visualizations using Seaborn.
- Create interactive dashboards using Plotly.
- Analyze customer purchasing patterns.
- Identify top-performing products and regions.
- Export reports and visualizations for business decision-making.

# Dataset Information

The dataset contains the following fields:

| Column | Description |
|---------|-------------|
| Date | Sales transaction date |
| Product | Product name |
| Quantity | Number of units sold |
| Price | Unit price |
| Customer_ID | Unique customer identifier |
| Region | Sales region |
| Total_Sales | Total transaction value |

# Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Interactive-Sales-Dashboard.git
```

Move into the project directory:

```bash
cd Interactive-Sales-Dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```
